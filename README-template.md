### What I learned

When I started making QR code component project I used h2 element for heading without knowing h1 hierarchy but later I came to know that it is useful for SEO, accessibility, screen readers and importance of semantic HTML. I first used section element to wrap everything inside it but I learned that using div element is great choice so I changed it to div. I also learned that we can use id element one per page or with javascript and with unique anchors but we can use class element multiple times. I used fixed width and height for flex container but it was not a good option to choose so I learned about viewport height and viewport width and changed it to height 100vh. I also added width 100vw then with the help of AI mentor I learned that browser will automatically fill up the width according to height so I kept only height 100vh.

### Continued development

I want to get better with other CSS properties for height and width and also want to learn more about complex flexbox and grid layouts.
